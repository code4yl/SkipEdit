SkipEdit

SkipEdit is a tool-in-progress that helps content creators turn raw videos into short-form, optimized clips for TikTok, Reels, and YouTube Shorts — with the help of AI.

✨ Goals:
- Automatically add captions, hooks, and callouts to videos
- Let users choose from 3 thumbnail options
- Help creators post faster without needing a full video editor

🔧 Built With:
- HTML / CSS (UI Prototype)
- Replit (Code + Testing)
- ChatGPT (AI logic & prompting help)
- Planned AI features (captioning, CTA insertions, content scanners)

📌 Status:
🚧 Still under construction. This is a personal learning project to explore AI + automation tools for content creators.

---

🙋‍♂️ Why I Built It:
I’ve seen how hard it is for small creators to post consistently. Editing takes time. This tool is my solution — something I wish existed when I started creating.
